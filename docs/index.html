Obsidian Plugin

Mandala Chart

大谷翔平で有名な曼荼羅チャートを、Obsidianでインタラクティブに作成・編集できるプラグイン。9×9グリッドで目標を構造化しよう。

GitHub → Obsidianでインストール

主な機能

シンプルなクリック操作で、すべてを直感的に。

🎯

インタラクティブ9×9グリッド

セルをクリックするだけで編集。全81セルが画面いっぱいに広がります。

🔍

2つの表示モード

全体表示でチャート全体を把握し、テーマ別詳細表示で深掘りできます。

🌈

虹色グラデーション

8テーマが時計回りに赤→橙→黄緑→緑→青緑→青→紫→ピンクと並びます。

⭕

円形デザイン

中央のテーマラベルはすべて美しい円形で表示。視覚的に区別しやすい。

🖨

印刷対応

ボタン1つでA4横向きに最適化した印刷ページが開きます。

🌐

5言語対応

Obsidianの言語設定を自動検出。日・英・中・韓・西に対応。

📌

リボンアイコン

左サイドバーのアイコンをクリックするだけで専用タブが開きます。

📝

ノート埋め込み

```mandalaコードブロックで、どのノートにも埋め込めます。

🌈 虹色カラーデザイン

8テーマが時計回りに虹のグラデーションを形成。中央は黄色で際立ちます。

📦 インストール

3ステップで使い始められます。

Obsidianの設定を開く

設定 → コミュニティプラグイン を開き、セーフモードをオフにします。

「Mandala Chart」を検索

「参照」をクリックして Mandala Chart を検索し、インストールします。

有効化して完了！

プラグインを有効化すると、左サイドバーにアイコンが表示されます。
